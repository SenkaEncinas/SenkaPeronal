Política de Privacidad

AsistentePersonal · Última actualización: junio 2026

¿Qué es AsistentePersonal?

AsistentePersonal es un bot de WhatsApp de uso privado que permite controlar dispositivos del hogar (luces via ESP32/MQTT) y funciona como asistente personal con acceso a Google Calendar, Google Tasks, información de clima y precios de criptomonedas. El acceso está restringido a usuarios autorizados.

Datos que procesamos

  • Número de teléfono de WhatsApp del usuario
  • Contenido de los mensajes de texto enviados al bot
  • Archivos de audio enviados (para transcripción)
  • Datos de Google Calendar y Google Tasks (solo del propietario autorizado)

Cómo usamos los datos

  • Procesar comandos de control del hogar (encender/apagar luces)
  • Consultar y crear eventos en Google Calendar
  • Consultar y agregar tareas en Google Tasks
  • Transcribir mensajes de voz mediante Groq Whisper API
  • Verificar permisos de acceso (ADMIN, FAMILIA, AMIGOS)

Almacenamiento y retención

Los mensajes son procesados en tiempo real y no se almacenan en bases de datos permanentes. Los estados de conversación se mantienen temporalmente en memoria del servidor para flujos de múltiples pasos (ej: crear un evento). Los tokens de autenticación de Google se almacenan de forma segura como archivos secretos en el servidor.

Terceros

  • Meta / WhatsApp Business API — entrega de mensajes
  • Google Calendar & Tasks API — gestión de agenda
  • Groq API (Whisper) — transcripción de audio
  • HiveMQ Cloud — broker MQTT para control de dispositivos
  • Render.com — infraestructura del servidor
  • wttr.in — datos de clima
  • Binance P2P — precios de criptomonedas

Acceso y control

Solo usuarios explícitamente autorizados por el administrador pueden interactuar con el bot. No recopilamos datos de menores de edad. No vendemos ni compartimos datos con terceros fuera de los servicios listados arriba.
